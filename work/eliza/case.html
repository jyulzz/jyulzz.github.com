Context

Eliza was here1 is an off-the-beaten path travel brand.

Its core customer type is female, 25 to 35 years old, college-educated and urban-dwelling.

Most own smartphones and expect to be able to use an application to interact with the brand both on and off location, to have access their booking information, see a timeline of their holiday's events, check accomodation information, local area maps, and more.

Due to design flaws, the previous version of the Eliza was here application had to be pulled from mobile application stores. A new version needed to be shipped as soon as possible.

To speed up the process, it was decided to use the application made for another brand of the group, Sunweb, as a foundation for this project.

Summary

Role

  • Designer within a Scrum team.

Deliverable

An MVP2 release of a mobile application for the off-the-beaten-path travel brand Eliza was here, using the existing Sunweb mobile application as a base.

Team

  • 11 developers,
  • 1 designer (me),
  • 1 product manager.

Results

  • A new version of the Eliza was here application was delivered in record time
  • Highly positive feedback from Eliza was here brand managers
  • 4.5/5 user rating in the Apple App Store and 4.9/5 user rating in the Google Play Store

This redesign started fresh, without re-using elements from the previous version of the application.

To get inspiration, I conducted a Style Inventory of the brand's main product, the Eliza was here website.

Final Thoughts

Having time and scope constraints always makes for an interesting design challenge. When an application has had to be pulled, the right way to serve users is delivering an improved replacement as quickly as possible. Without the solid foundation of the Sunweb application both in terms of design assets and code base, it would have been difficult to push a first release so quickly. Producing quality first always pays off later, often in unexpected ways. Ultimately, the success of this project reinforced my belief in a pragmatic approach, the value of the agile process in this type of situation and the importance of producing high quality products that can be extended at a lower cost down the line.
